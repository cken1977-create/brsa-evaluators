# BRSA Evaluator Portal

The BRSA Evaluator Portal is the official web interface for evaluator training, certification, and evaluation operations within the Behavioral Readiness Standards Authority (BRSA) ecosystem.

This portal connects to the BRSA backend infrastructure and allows certified evaluators to:

• Complete evaluator training modules  
• Take certification assessments  
• Access case simulation training  
• Review assigned evaluations  
• Submit readiness evaluations  
• Upload supporting evidence  
• View evaluation history and audit trails

## Purpose

The BRSA Evaluator Portal supports the professional evaluator workforce responsible for conducting behavioral readiness evaluations under BRSA standards.

The portal ensures evaluators operate under consistent protocols, documented evidence standards, and auditable evaluation procedures.
